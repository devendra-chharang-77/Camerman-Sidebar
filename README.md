# Camerman-Sidebar
# overview:-
SideGlide is a modern sidebar navigation menu built using HTML & CSS. It includes smooth animations, hover effects, and stylish icons to enhance user experience. The sidebar is responsive and works seamlessly across different screen sizes.
 
 # Features
  Features
(1) Stylish Sidebar Menu – Aesthetic and smooth navigation panel
(2) Hover Effects – Interactive and visually appealing
(3) Social Media Links – Quick access to social platforms
(4) Icons Integrated – Uses FontAwesome for better UI
(5) Fully Responsive – Works on all devices

# Technologies Used
HTML5 – Structure of the project
CSS3 – Styling & animations
FontAwesome – Icons
